Insert order

    This button uses for insert new order to the database.

  1. Choose client

    You can add new client for this you should add all information about client. Telephone write in format 00420XXXXXX, name, surname, model and brand should start from capital letter, license is one capital letter. Then after you safe order this client will be added to clients database.

  2. Choose vehicle

    You can add new vehicle to existing client. If you chose new client before you should add new vehicle.

  3. Choose date

    Choose only date in the future.

  4. Choose time from and time to

    Time to should be later than time from.

  5. Choose driver

    The driver license should compare with vehicle license, otherwise you couldn't save order.

  6. Choose driving a person

    Make the check if in car will be someone else except driver.

  7. Choose place from and place to

  8. Write some comments

Insert driver

    Write all information about driver. Telephone write in format 00420XXXXXX, name and surname should start from capital letter, license is one capital letter, time from and time to is time when driver has time for work.

Show orders

    In this table will be all existing order. You can choose one and delete it.
